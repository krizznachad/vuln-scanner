# 🔍 VulnReconTool v1.0

A simple vulnerability scanner built with Python.  
It performs:

- ✅ Port scanning (1–1024)
- ✅ HTTP header analysis (security header check)
- ✅ Directory brute-force (finds hidden/admin paths)

## 🛠 Modules Used

- `socket`
- `requests`
- `threading`
- `colorama`
- `queue`
